# Fashion Store Mobile Application – Manual Testing

## Overview
This repository contains manual testing documentation for a Fashion Store mobile application.

## Purpose
This project was created independently to **improve manual software testing skills** through hands‑on practice on a real‑world mobile application.

## Project Background
The Fashion Store mobile application was developed independently by another individual.  
My contribution to this project was limited to **manual testing activities only**, performed to strengthen my QA and testing skills.

## Role & Responsibilities
As a self‑learning manual tester, I was responsible for:
- Understanding the application functionality
- Designing test scenarios and manual test cases
- Executing test cases
- Identifying and documenting defects
- Preparing test execution summary
- Creating a traceability matrix

## Scope of Testing
- Functional Testing
- UI Testing
- Usability Testing
- Basic Regression Testing

## Test Artifacts Included
- Test Plan  
- Test Scenarios  
- Test Cases  
- Defect Report  
- Test Execution Summary  
- Traceability Matrix
